> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the next demonstrable AuditX desktop milestone: a Tauri desktop window that can start the existing Phase 1A fake audit API flow from the React UI and display job/findings results.

**Architecture:** Keep the current FastAPI fake audit backend as the source of truth and connect the React desktop UI to it through a small typed API client. Do not add real OCR, LLM, PDF parsing, or Python sidecar packaging in this plan; the purpose is to turn the now-stable desktop shell into a clickable demo loop.

**Tech Stack:** Tauri 2, React, Vite, TypeScript, Python FastAPI, Pydantic, pytest, npm, cargo.

- [ ] **Step 2: Create the API client**

Create `frontend/src/api/auditJobs.ts`:

```ts
import type { AuditJob } from "../types/audit";

const API_BASE_URL = "http://127.0.0.1:8765";

async function requestJson<T>(path: string, init?: RequestInit): Promise<T> {
  const response = await fetch(`${API_BASE_URL}${path}`, {
    ...init,
    headers: {
      "Content-Type": "application/json",
      ...init?.headers,
    },
  });

  if (!response.ok) {
    const detail = await response.text();
    throw new Error(`Audit API request failed: ${response.status} ${detail}`);
  }

  return response.json() as Promise<T>;
}

export async function createAuditJob(filePath: string): Promise<AuditJob> {
  return requestJson<AuditJob>("/api/audit-jobs", {
    method: "POST",
    body: JSON.stringify({ file_path: filePath }),
  });
}
```

```ts
const DEMO_FILE_PATH = "demo_resume.pdf";

export function App() {
  const [job, setJob] = useState<AuditJob | null>(null);
  const [isRunning, setIsRunning] = useState(false);
  const [error, setError] = useState<string | null>(null);

  async function handleRunAudit() {
    setIsRunning(true);
    setError(null);

    try {
      const nextJob = await createAuditJob(DEMO_FILE_PATH);
      setJob(nextJob);
    } catch (auditError) {
      setError(auditError instanceof Error ? auditError.message : "Unknown audit error");
    } finally {
      setIsRunning(false);
    }
  }

  const findings = job?.findings ?? [];

  return (
    <main className="app-shell">
      {/* keep the current hero/workspace layout, but wire button/status/findings into it */}
    </main>
  );
}
```
